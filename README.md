# AAU辩论网站 - 现代化UI重新设计



## 技术栈
- HTML5
- CSS3（使用现代布局技术）
- 原生JavaScript
- SVG图表（无需外部图表库）

## 设计特色



### 设计原则
1. **现代简约风格** - 清晰的层次和留白
2. **玻璃态效果** - 半透明背景增加层次感
3. **统一圆角** - 16px主圆角，12px按钮，8px小元素
4. **柔和阴影** - 根据深度使用不同层级的阴影
5. **微交互动画** - 平滑的过渡和反馈效果

## 页面功能

### 1. 主仪表盘
- 统计数据展示卡片
- 赛事趋势图表
- 选手活跃度分析
- 近期活动时间线

### 2. 赛果管理
- 标签页式筛选器
- 赛事卡片网格布局
- 对阵双方比分展示
- 赛事状态标识

### 3. 财务管理
- 收支概览卡片
- 预算使用进度条
- 财务趋势图表
- 交易记录表格

### 4. 积分排名
- Top 3奖台式展示
- 积分趋势图表
- 详细排名列表
- 成就徽章系统

## 浏览器支持
- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+

## 使用说明

1. 直接在浏览器中打开 `index.html` 即可查看网站
2. 点击顶部导航栏切换不同页面
3. 所有交互效果均已实现
4. 图表使用SVG实现，无需外部依赖

## 特色功能

### 响应式设计
- 支持从480px到1440px的各种屏幕尺寸
- 移动端优化显示
- 平板端适配布局

### 交互动画
- 页面切换淡入效果
- 卡片悬浮动画
- 统计数字递增动画
- 按钮点击反馈

### 无障碍设计
- 色彩对比度符合WCAG标准
- 键盘导航支持
- 语义化HTML结构
